                                <p class="text-sm text-slate-600 leading-relaxed mb-2"><strong class="text-sky-700">Definition:</strong> The rate at which pressure increases with vertical depth. In well control, it is commonly expressed in <span class="font-mono bg-slate-100 px-1.5 py-0.5 rounded border border-slate-200">psi/ft</span> (API).</p>
                            </div>

                            <!-- 1. Normal Formation Pressure -->
                            <div class="flex flex-col md:flex-row gap-6 items-center bg-white p-5 rounded-xl border border-slate-200 shadow-sm">
                                <div class="relative w-24 h-32 bg-slate-100 border-2 border-slate-300 rounded flex-shrink-0 overflow-hidden flex flex-col justify-end shadow-inner">
                                    <div class="w-full h-full bg-sky-300/30 absolute bottom-0"></div>
                                    <div class="w-full h-1/2 bg-sky-500 relative z-10 flex items-center justify-center text-[10px] text-white font-bold text-center leading-tight">Native<br>Fluid</div>
                                    <div class="absolute top-2 w-full text-center text-sky-800 font-bold opacity-40">↓</div>
                                </div>
                                <div class="flex-grow">
                                    <h4 class="font-bold text-sky-700 text-lg mb-1 flex items-center gap-2"><span class="text-sky-400">🌊</span> 1. Normal Formation Pressure</h4>
                                    <p class="text-xs text-slate-600 mb-2"><strong>IWCF Definition:</strong> A pressure equal to the hydrostatic pressure of a full column of native formation fluid extending from the depth of the formation to the surface.</p>
                                    <p class="text-xs text-slate-600 mb-3"><strong>The Mechanism:</strong> The pore spaces are connected all the way to the surface (open system). The fluids within the pores only support the weight of the fluid column above them.</p>
                                    <div class="grid grid-cols-1 sm:grid-cols-2 gap-3 text-[10px] text-slate-700">
                                        <div class="bg-slate-50 p-2 rounded border border-slate-100">
                                            <strong class="block mb-1 text-slate-800">Standard IWCF Values:</strong>
                                            <ul class="space-y-0.5">
                                                <li>• <strong>Freshwater:</strong> 0.433 psi/ft (8.33 ppg)</li>
                                                <li>• <strong>Saltwater:</strong> 0.465 psi/ft (8.94 ppg)</li>
                                            </ul>
                                        </div>
                                        <div class="bg-sky-50 p-2 rounded border border-sky-100">
                                            <strong class="block mb-1 text-sky-800">Operational Rule:</strong>
                                            Drill safely with MW slightly above native (e.g., 9.0 - 9.2 ppg) to provide a standard trip margin.
                                        </div>
                                    </div>
                                </div>
                            </div>

                            <!-- 2. Abnormal (Overpressure) -->
                            <div class="flex flex-col md:flex-row gap-6 items-center bg-rose-50 p-5 rounded-xl border border-rose-200 shadow-sm">
                                <div class="relative w-24 h-32 bg-slate-100 border-2 border-slate-300 rounded flex-shrink-0 overflow-hidden flex flex-col justify-end shadow-inner">
                                    <div class="w-full h-8 bg-slate-700 absolute top-4 border-b-4 border-slate-900 z-20 flex items-center justify-center text-[10px] text-white font-bold shadow-md">Cap Rock</div>
                                    <div class="w-full h-20 bg-rose-500 relative z-10 flex items-center justify-center text-[10px] text-white font-bold text-center leading-tight anim-pulse-pressure">Trapped<br>Fluid</div>
                                    <div class="absolute bottom-2 w-full flex justify-around px-2 z-20">
                                        <span class="text-white font-bold text-sm animate-pulse">↑</span>
                                        <span class="text-white font-bold text-sm animate-pulse">↑</span>
                                    </div>
                                </div>
                                <div class="flex-grow">
                                    <h4 class="font-bold text-rose-700 text-lg mb-1 flex items-center gap-2"><span class="text-rose-400">💥</span> 2. Abnormal Formation Pressure (Overpressure)</h4>
                                    <p class="text-xs text-slate-700 mb-2"><strong>IWCF Definition:</strong> Any formation pore pressure that is greater than the normal hydrostatic pressure of the formation water for that specific TVD.</p>
                                    <p class="text-xs text-slate-700 mb-3"><strong>The Mechanism:</strong> Requires a <em>sealing mechanism</em> (cap rock or impermeable barrier) and a <em>pressurizing mechanism</em>. Trapped fluid is forced to support a portion of the overlying rock matrix (the overburden).</p>
                                    <div class="text-[10px] text-slate-800 bg-white p-3 rounded border border-rose-100">
                                        <strong class="block mb-1 text-rose-800 border-b border-rose-50 pb-1">Primary Causes (IWCF Syllabus Focus):</strong>
                                        <ul class="grid grid-cols-1 sm:grid-cols-2 gap-x-2 gap-y-1 mt-1 list-disc pl-4">
                                            <li><strong>Under-compaction of Shales</strong> (Most Common): Trapped water bears rock weight.</li>
                                            <li><strong>Artesian Effect:</strong> Formation outcrop is higher than rig floor.</li>
                                            <li><strong>Gas Cap Effect:</strong> Low-density gas transmits high bottom pressure to the top.</li>
                                            <li><strong>Faulting / Salt Domes:</strong> Fluid migration or blocked pathways.</li>
                                        </ul>
                                    </div>
                                </div>
                            </div>

                            <!-- 3. Subnormal Pressure -->
                            <div class="flex flex-col md:flex-row gap-6 items-center bg-amber-50 p-5 rounded-xl border border-amber-200 shadow-sm">
                                <div class="relative w-24 h-32 bg-slate-100 border-2 border-slate-300 rounded flex-shrink-0 overflow-hidden flex flex-col justify-end shadow-inner">
                                    <div class="w-full h-10 bg-amber-700/40 relative z-10 flex items-center justify-center text-[10px] text-amber-900 font-bold text-center leading-tight">Depleted<br>Zone</div>
                                    <div class="absolute bottom-1 w-full flex justify-around px-2 z-20 opacity-50">
                                        <span class="text-amber-900 font-bold text-sm">↓</span>
                                        <span class="text-amber-900 font-bold text-sm">↓</span>
                                    </div>
                                </div>
                                <div class="flex-grow">
                                    <h4 class="font-bold text-amber-700 text-lg mb-1 flex items-center gap-2"><span class="text-amber-500">📉</span> 3. Subnormal Formation Pressure</h4>
                                    <p class="text-xs text-slate-700 mb-3">Pore pressure that is less than the normal hydrostatic pressure for that vertical depth.</p>
                                    <div class="text-[10px] text-slate-800 bg-white p-3 rounded border border-amber-100 mb-3">
                                        <strong class="block mb-1 text-amber-800 border-b border-amber-50 pb-1">Primary Causes:</strong>
                                        <ul class="list-disc pl-4 mt-1 space-y-0.5">
                                            <li><strong>Depletion:</strong> The most common cause in modern drilling. Hydrocarbons produced from reservoir artificially lower the pressure.</li>
                                            <li><strong>Thermal Contraction:</strong> Cooling of formation fluids.</li>
                                        </ul>
                                    </div>
                                    <div class="bg-rose-100 text-rose-800 p-2 rounded text-[11px] border border-rose-200 flex items-center gap-2 font-medium">
                                        <span class="text-lg">⚠️</span> <span><strong>Operational Risk:</strong> Severe lost circulation. Standard mud weights will be heavily overbalanced.</span>
                                    </div>
                                </div>
                            </div>
                            
                            <!-- Conclusion Reference -->
                            <div class="w-full bg-slate-800 p-5 rounded-xl border border-slate-700 shadow-sm mt-2">
                                <h4 class="text-sm font-bold text-sky-400 uppercase tracking-widest mb-3 border-b border-slate-700 pb-2 flex items-center gap-2"><span>📌</span> Gradient Quick Reference Rule (Saltwater Base)</h4>
                                <div class="grid grid-cols-1 sm:grid-cols-3 gap-4 text-center">
                                    <div class="bg-slate-700/50 p-3 rounded-lg border border-slate-600 flex flex-col justify-center">
                                        <span class="block text-[11px] uppercase tracking-wider text-amber-400 font-bold mb-1">Subnormal</span>
                                        <span class="font-mono text-white text-sm font-semibold">FP &lt; 0.465 psi/ft</span>
                                    </div>
                                    <div class="bg-slate-700/50 p-3 rounded-lg border border-slate-600 flex flex-col justify-center ring-1 ring-sky-500/50">
                                        <span class="block text-[11px] uppercase tracking-wider text-sky-400 font-bold mb-1">Normal</span>
                                        <span class="font-mono text-white text-sm font-semibold">FP = 0.465 psi/ft</span>
                                    </div>
                                    <div class="bg-slate-700/50 p-3 rounded-lg border border-slate-600 flex flex-col justify-center">
                                        <span class="block text-[11px] uppercase tracking-wider text-rose-400 font-bold mb-1">Abnormal</span>
                                        <span class="font-mono text-white text-sm font-semibold">FP &gt; 0.465 psi/ft</span>
                                    </div>
                                </div>
                            </div>
                        </div>

                        <!-- Panel: Fracture Pressure -->
                        <div id="def-frac" class="def-panel p-8 h-full flex flex-col md:flex-row gap-8 items-center hidden custom-scroll overflow-y-auto">
                            <div class="relative w-32 h-48 bg-amber-900 border-2 border-slate-300 rounded flex-shrink-0 overflow-hidden flex flex-col items-center justify-center">
                                <div class="w-12 h-full bg-sky-500 relative z-10 flex items-center justify-center">
                                    <div class="text-white font-bold text-xl anim-pulse-pressure rounded-full px-1">💥</div>
                                </div>
                                <!-- Animated Cracks -->
                                <div class="absolute top-[40%] left-0 w-[50%] h-1 bg-sky-400 anim-rock-crack origin-right"></div>
                                <div class="absolute top-[70%] right-0 w-[50%] h-1 bg-sky-400 anim-rock-crack origin-left" style="animation-delay: 0.5s"></div>
                            </div>
                            <div>
                                <h3 class="text-2xl font-bold text-slate-900 border-b border-slate-200 pb-2 mb-4">Fracture Pressure</h3>
                                <p class="text-slate-600 leading-relaxed mb-4">The pressure at which the formation rock matrix breaks or splits, allowing wellbore fluids to enter the induced fractures. Exceeding this pressure causes severe lost circulation.</p>
                                <span class="bg-rose-100 text-rose-800 text-xs font-bold px-3 py-1.5 rounded uppercase tracking-wider">Upper Operating Limit</span>
                            </div>
                        </div>

                        <!-- Panel: Overburden Pressure -->
                        <div id="def-overburden" class="def-panel p-8 h-full flex flex-col md:flex-row gap-8 items-center hidden custom-scroll overflow-y-auto">
                            <div class="relative w-32 h-48 border-2 border-slate-300 rounded flex-shrink-0 overflow-hidden flex flex-col">
                                <div class="h-1/3 bg-slate-300 border-b-2 border-slate-400 w-full flex justify-center items-end pb-1 text-slate-500 font-bold text-lg">↓</div>
                                <div class="h-1/3 bg-slate-500 border-b-2 border-slate-600 w-full flex justify-center items-end pb-1 text-slate-700 font-bold text-lg">↓</div>
                                <div class="h-1/3 bg-amber-900 w-full flex justify-center items-end pb-1 text-amber-700 font-bold text-lg">↓</div>
                                <div class="absolute inset-0 flex items-center justify-center opacity-40">
                                    <div class="w-2 h-32 bg-slate-800"></div>
                                    <div class="absolute bottom-6 w-0 h-0 border-l-[12px] border-r-[12px] border-t-[20px] border-transparent border-t-slate-800"></div>
                                </div>
                            </div>
                            <div>
                                <h3 class="text-2xl font-bold text-slate-900 border-b border-slate-200 pb-2 mb-4">Overburden Pressure</h3>
                                <p class="text-slate-600 leading-relaxed">The pressure exerted by the total weight of the overlying rock matrix and the fluids contained within its pore spaces at a specific depth. It represents the absolute maximum stress at any given point.</p>
                            </div>
                        </div>

                        <!-- Panel: Static vs Dynamic BHP -->
                        <div id="def-bhp" class="def-panel p-8 h-full flex flex-col gap-6 hidden custom-scroll overflow-y-auto">
                            <div class="space-y-4">
                                <h3 class="text-2xl font-bold text-slate-900 border-b border-slate-200 pb-2">Bottom Hole Pressure (BHP)</h3>
                                <p class="text-sm text-slate-600 mb-2">BHP is the total pressure exerted at the bottom of the wellbore. Its value changes depending on whether the fluid is static or moving.</p>
                            </div>
                            <div class="grid grid-cols-1 md:grid-cols-2 gap-6">
                                <div class="bg-slate-50 p-4 rounded-xl border border-slate-200 shadow-sm relative overflow-hidden">
                                    <div class="absolute top-2 right-2 text-2xl opacity-20">🛑</div>
                                    <strong class="text-sky-700 block mb-1">Static BHP:</strong>
                                    <p class="text-sm text-slate-600 mb-3">The pressure at the bottom of the well when the mud pumps are off and the fluid is not moving.</p>
                                    <div class="bg-white border border-slate-200 p-2 rounded text-xs font-mono text-slate-700 font-semibold inline-block">BHP = HP</div>
                                </div>
                                <div class="bg-sky-50 p-4 rounded-xl border border-sky-200 shadow-sm relative overflow-hidden">
                                    <div class="absolute top-2 right-2 text-2xl opacity-20 anim-flow-down">🌊</div>
                                    <strong class="text-sky-800 block mb-1">Dynamic BHP (Circulating):</strong>
                                    <p class="text-sm text-slate-600 mb-3">The pressure at bottom when pumps are on. Fluid friction moving up the annulus adds pressure.</p>
                                    <div class="bg-white border border-sky-200 p-2 rounded text-xs font-mono text-sky-800 font-semibold inline-block">BHP = HP + Annular Pressure Loss</div>
                                </div>
                            </div>
                        </div>

                        <!-- Panel: Choke vs Bell Nipple -->
                        <div id="def-choke" class="def-panel p-8 h-full flex flex-col gap-6 hidden custom-scroll overflow-y-auto">
                            <div class="space-y-2">
                                <h3 class="text-2xl font-bold text-slate-900 border-b border-slate-200 pb-2">The Choke vs. Bell Nipple Difference</h3>
                            </div>
                            <div class="grid grid-cols-1 md:grid-cols-2 gap-6">
                                <!-- Bell Nipple -->
                                <div class="bg-white p-4 rounded-xl border border-slate-200 shadow-sm flex flex-col items-center text-center">
                                    <div class="relative w-16 h-24 bg-slate-100 border-x-2 border-b-2 border-slate-300 rounded-b overflow-hidden mb-3">
                                        <div class="absolute bottom-0 w-full h-12 bg-sky-500"></div>
                                        <div class="absolute top-4 w-full h-12 bg-sky-400 anim-geyser opacity-80 blur-[1px]"></div>
                                    </div>
                                    <strong class="text-sky-700 block mb-1">Circulating to Bell Nipple</strong>
                                    <span class="text-[10px] font-bold bg-slate-200 text-slate-600 px-2 py-0.5 rounded uppercase tracking-wide mb-2">Open System</span>
                                    <p class="text-xs text-slate-600">The fluid returns to surface at atmospheric pressure. Dynamic BHP is solely HP plus Annular Friction.</p>
                                </div>
                                <!-- Choke -->
                                <div class="bg-rose-50 p-4 rounded-xl border border-rose-200 shadow-sm flex flex-col items-center text-center">
                                    <div class="relative w-16 h-24 bg-slate-100 border-x-2 border-b-2 border-slate-300 rounded-b overflow-hidden mb-3">
                                        <div class="absolute top-0 w-full h-4 bg-slate-800 z-20"></div> <!-- BOP/Choke -->
                                        <div class="absolute bottom-0 w-full h-full bg-sky-500"></div>
                                        <div class="absolute top-6 left-1/2 -translate-x-1/2 text-rose-200 font-bold text-xl anim-choke-restrict z-10">↓↓</div>
                                    </div>
                                    <strong class="text-rose-700 block mb-1">Circulating through Choke</strong>
                                    <span class="text-[10px] font-bold bg-rose-200 text-rose-800 px-2 py-0.5 rounded uppercase tracking-wide mb-2">Closed System</span>
                                    <p class="text-xs text-slate-700">Returning fluid is forced through the choke, applying backpressure. <br><strong>Dynamic BHP = HP + Friction + Choke Backpressure.</strong></p>
                                </div>
                            </div>
                        </div>

                        <!-- Panel: Surface Pressure Limits -->
                        <div id="def-limits" class="def-panel p-8 h-full flex flex-col gap-6 hidden custom-scroll overflow-y-auto">
                            <h3 class="text-2xl font-bold text-slate-900 border-b border-slate-200 pb-2">Well Control Surface Pressure Limits</h3>
                            <div class="space-y-5">
                                <div>
                                    <strong class="text-rose-600 block text-lg flex items-center gap-2"><span>💥</span> MAASP</strong>
                                    <span class="text-xs font-bold text-slate-500 block mb-1">Maximum Allowable Annular Surface Pressure</span>
                                    <p class="text-sm text-slate-600 mb-2">The maximum pressure that can be applied to the annulus at the surface (with the BOP closed) before the formation at the casing shoe breaks down. Assumes a static fluid column.</p>
                                    <div class="bg-rose-50 border border-rose-100 p-2 rounded text-xs font-mono text-rose-800 inline-block font-semibold">MAASP = (Frac MW - Current MW) × 0.052 × Shoe TVD</div>
                                </div>
                                <div class="border-t border-slate-100 pt-4">
                                    <strong class="text-amber-600 block text-lg flex items-center gap-2"><span>⚠️</span> MASP</strong>
                                    <span class="text-xs font-bold text-slate-500 block mb-1">Maximum Anticipated Surface Pressure</span>
                                    <p class="text-sm text-slate-600">The highest pressure predicted to be encountered at surface during a specific well operation (e.g., a worst-case gas kick reaching surface). Heavily used in equipment design and selection.</p>
                                </div>
                                <div class="border-t border-slate-100 pt-4">
                                    <strong class="text-slate-800 block text-lg flex items-center gap-2"><span>⚙️</span> MAWP</strong>
                                    <span class="text-xs font-bold text-slate-500 block mb-1">Maximum Allowable Working Pressure</span>
                                    <p class="text-sm text-slate-600">The maximum safe working pressure of a specific piece of equipment (e.g., BOP, choke manifold, valves) as rated by the manufacturer. Equipment must never be operated above its MAWP.</p>
                                </div>
                            </div>
                        </div>

                        <!-- Panel: Circulating Friction Losses -->
                        <div id="def-friction" class="def-panel p-8 h-full flex flex-col md:flex-row gap-8 items-center hidden custom-scroll overflow-y-auto">
                            <div class="relative w-32 h-64 bg-slate-100 border-2 border-slate-300 rounded-b-xl flex-shrink-0 flex items-center justify-center overflow-hidden shadow-inner">
                                <div class="absolute bottom-0 w-full h-[90%] bg-sky-500 opacity-50"></div>
                                <div class="absolute top-0 w-8 h-[80%] bg-slate-400 border-x-2 border-slate-500"></div>
                                <!-- BHA -->
                                <div class="absolute top-[80%] w-10 h-[10%] bg-slate-600 border-x-2 border-slate-700"></div>
                                <!-- Bit High Friction -->
                                <div class="absolute top-[90%] w-12 h-2 anim-pulse-friction rounded-full"></div>
                                <!-- Annulus Friction -->
                                <div class="absolute right-2 top-[30%] w-3 h-32 bg-orange-400 opacity-40 blur-[2px] rounded-full"></div>
                                <div class="absolute left-2 top-[30%] w-3 h-32 bg-orange-400 opacity-40 blur-[2px] rounded-full"></div>
                            </div>
                            <div class="space-y-4 w-full">
                                <h3 class="text-2xl font-bold text-slate-900 border-b border-slate-200 pb-2 mb-2">Circulating Friction Losses</h3>
                                <p class="text-sm text-slate-600 mb-4">During circulation, mud pump pressure is entirely consumed by friction as fluid travels through the system.</p>
                                
                                <div class="space-y-2">
                                    <div class="bg-white p-3 rounded shadow-sm border border-slate-200">
                                        <strong class="text-sky-700 text-sm block">APL (Annular Pressure Loss)</strong>
                                        <p class="text-xs text-slate-600">Pressure required to push fluid up the annulus from bit to surface.</p>
                                    </div>
                                    <div class="bg-white p-3 rounded shadow-sm border border-slate-200">
                                        <strong class="text-slate-700 text-sm block">Surface Losses</strong>
                                        <p class="text-xs text-slate-600">Friction through standpipe, kelly hose, top drive, and surface piping.</p>
                                    </div>
                                    <div class="bg-white p-3 rounded shadow-sm border border-slate-200">
                                        <strong class="text-slate-700 text-sm block">Drill String & BHA Losses</strong>
                                        <p class="text-xs text-slate-600">Friction through the inside of drill pipe, HWDP, drill collars, and tools.</p>
                                    </div>
                                    <div class="bg-white p-3 rounded shadow-sm border border-rose-200 border-l-4 border-l-rose-500">
                                        <strong class="text-rose-700 text-sm block">Bit Losses</strong>
                                        <p class="text-xs text-slate-600">The massive pressure drop across the bit nozzles to maximize hydraulic impact.</p>
                                    </div>
                                </div>
                            </div>
                        </div>

                        <!-- Panel: Key Distinctions -->
                        <div id="def-distinctions" class="def-panel p-8 h-full flex flex-col gap-6 hidden custom-scroll overflow-y-auto">
                            <h3 class="text-2xl font-bold text-slate-900 border-b border-slate-200 pb-2">Key Distinctions in the System</h3>
                            <div class="space-y-6">
                                <div class="bg-rose-50 p-6 rounded-xl border border-rose-100 shadow-sm relative">
                                    <div class="absolute -top-3 -left-3 text-4xl">🥇</div>
                                    <strong class="text-rose-800 block text-lg mb-2 pl-4">Where is the highest friction loss?</strong>
                                    <p class="text-sm text-slate-700 font-medium pl-4">➔ The Bit.</p>
                                    <p class="text-sm text-slate-600 mt-2 pl-4">A significant portion of the total pump pressure (often 50% or more) is intentionally dropped across the bit nozzles to maximize hydraulic impact and clean the hole.</p>
                                </div>
                                <div class="bg-sky-50 p-6 rounded-xl border border-sky-100 shadow-sm relative">
                                    <div class="absolute -top-3 -left-3 text-4xl">🎯</div>
                                    <strong class="text-sky-800 block text-lg mb-2 pl-4">Which part is most important for well control?</strong>
                                    <p class="text-sm text-slate-700 font-medium pl-4">➔ The Annulus (APL).</p>
                                    <p class="text-sm text-slate-600 mt-2 pl-4">While it accounts for a very small percentage of the total system pressure loss, it acts directly on the open hole. APL increases the dynamic BHP, creating Equivalent Circulating Density (ECD). If APL is too high, it can fracture the formation; if suddenly removed (shutting off pumps), the pressure drop can allow a kick.</p>
                                </div>
                            </div>

        function drawWellbore(isKicking, pumpsOn, env, spm) {
            if (!ctxSim || !wellCanvas) return;
            
            ctxSim.clearRect(0, 0, wellCanvas.width, wellCanvas.height);
            let holeW = env === 'slim' ? 60 : 80;
            const holeX = (wellCanvas.width - holeW) / 2;
            const formY = wellCanvas.height - 120;

            ctxSim.fillStyle = '#92400e'; 
            ctxSim.fillRect(0, formY, wellCanvas.width, wellCanvas.height - formY);
            
            ctxSim.fillStyle = '#334155'; 
            ctxSim.fillRect(holeX, 0, holeW, wellCanvas.height);

            ctxSim.fillStyle = isKicking ? 'rgba(244, 63, 94, 0.4)' : '#0284c7'; 
            ctxSim.fillRect(holeX + 6, 0, holeW - 12, wellCanvas.height);

            const dpW = 16;
            ctxSim.fillStyle = '#94a3b8'; 
            ctxSim.fillRect(holeX + (holeW/2) - (dpW/2), 0, dpW, wellCanvas.height);

            if(pumpsOn && !isKicking && spm > 0) {
                ctxSim.fillStyle = 'rgba(255,255,255,0.4)';
                let flowSpeed = Math.max(1, spm / 25);
                pumpFlowOffset += flowSpeed;
                if(pumpFlowOffset > 40) pumpFlowOffset = 0;
                for(let i = -40; i < wellCanvas.height; i += 40) {
                    ctxSim.fillRect(holeX + (holeW/2) - 2, i + pumpFlowOffset, 4, 10);
                    ctxSim.fillRect(holeX + 10, wellCanvas.height - (i + pumpFlowOffset), 3, 10);
                    ctxSim.fillRect(holeX + holeW - 13, wellCanvas.height - (i + pumpFlowOffset), 3, 10);
                }
            }

            if (isKicking) {
                let bubbleSpeedMult = env === 'slim' ? 2 : 1; 
                if(bubbles.length < 60) {
                    bubbles.push({ x: holeX + 10 + Math.random() * (holeW - 20), y: wellCanvas.height, r: Math.random() * 5 + 2, s: (Math.random() * 4 + 3) * bubbleSpeedMult });
                }
                ctxSim.fillStyle = 'rgba(255, 255, 255, 0.7)';
                for (let i = bubbles.length - 1; i >= 0; i--) {
                    let b = bubbles[i];
                    ctxSim.beginPath(); ctxSim.arc(b.x, b.y, b.r, 0, Math.PI * 2); ctxSim.fill();
                    b.y -= b.s; b.x += Math.sin(b.y * 0.1) * 1.5;
                    if (b.y < -10) bubbles.splice(i, 1);
                }
            } else {
                bubbles = [];
            }
            
            if(isKicking || pumpsOn) {
                cancelAnimationFrame(simAnimFrame);
                simAnimFrame = requestAnimationFrame(() => drawWellbore(isKicking, pumpsOn, env, spm));
            } else {
                cancelAnimationFrame(simAnimFrame);
            }
        }

        let timerInterval;
        let timeRemaining = 90 * 60; // 1.5 hours in seconds

        function updateTimerDisplay() {
            const minutes = Math.floor(timeRemaining / 60);
            const seconds = timeRemaining % 60;
            const display = `${minutes.toString().padStart(2, '0')}:${seconds.toString().padStart(2, '0')}`;
            const timerEl = document.getElementById('quiz-timer');
            if(timerEl) {
                timerEl.textContent = display;
                if(timeRemaining < 300) { 
                    timerEl.classList.remove('text-amber-400');
                    timerEl.classList.add('text-rose-500', 'animate-pulse');
                } else {
                    timerEl.classList.add('text-amber-400');
                    timerEl.classList.remove('text-rose-500', 'animate-pulse');
                }
            }
        }

        window.startQuiz = function() {
            const nameInput = val('student-name')?.trim();
            const posInput = val('student-position');
            const expInput = val('student-exp');

            if(!nameInput || !posInput || expInput === "") return alert("Please fill in all details, including years of experience.");
            
            activeStudentData = { name: nameInput, role: posInput, exp: parseInt(expInput) };
            setTxt('active-student-display', `Student: ${activeStudentData.name} | Role: ${activeStudentData.role} (${activeStudentData.exp} yrs)`);
            
            // Randomly shuffle the questions for this attempt
            activeQuizPool = [...quizQuestions].sort(() => Math.random() - 0.5);

            currentQuizQuestion = 0; quizScore = 0; currentAttemptData = []; setTxt('current-score', '0');
            document.getElementById('quiz-start')?.classList.add('hidden'); document.getElementById('quiz-results')?.classList.add('hidden'); document.getElementById('quiz-question')?.classList.remove('hidden');
            
            timeRemaining = 90 * 60;
            updateTimerDisplay();
            clearInterval(timerInterval);
            timerInterval = setInterval(() => {
                timeRemaining--;
                updateTimerDisplay();
                if (timeRemaining <= 0) {
                    clearInterval(timerInterval);
                    alert("Time is up! Your assessment will now be submitted.");
                    window.showQuizResults();
                }
            }, 1000);

            loadQuestion();
        };

        function loadQuestion() {
            isAnswered = false; 
            document.getElementById('next-btn-container')?.classList.add('hidden');
            const expBox = document.getElementById('explanation-box');
            if (expBox) expBox.classList.add('hidden');

            const qData = activeQuizPool[currentQuizQuestion];
            setTxt('question-counter', `Question ${currentQuizQuestion + 1} of ${activeQuizPool.length}`); 
            setTxt('question-text', qData.q);
            
            const optsContainer = document.getElementById('options-container');
            if(optsContainer) {
                optsContainer.innerHTML = '';
                qData.options.forEach((opt, index) => {
                    const btn = document.createElement('button');
                    btn.className = 'w-full text-left p-4 rounded-xl border border-slate-600 bg-slate-800 text-slate-300 hover:bg-slate-700 hover:border-sky-500 transition-colors';
                    btn.textContent = opt; 
                    btn.onclick = () => window.checkAnswer(index, btn);
                    optsContainer.appendChild(btn);
                });
            }
        }

        window.checkAnswer = function(selectedIndex, btnElement) {
            if (isAnswered) return; isAnswered = true;
            const qData = activeQuizPool[currentQuizQuestion]; const allBtns = document.getElementById('options-container').querySelectorAll('button');
            const isCorrect = selectedIndex === qData.answer;
            
            currentAttemptData.push({ qIndex: currentQuizQuestion, isCorrect: isCorrect, topic: qData.topic });

            if(allBtns) {
                allBtns.forEach(btn => { btn.classList.remove('hover:bg-slate-700', 'hover:border-sky-500'); btn.classList.add('opacity-50', 'cursor-default'); });
            }

            if (isCorrect) {
                btnElement.classList.replace('bg-slate-800', 'bg-emerald-900'); btnElement.classList.replace('border-slate-600', 'border-emerald-500');
                btnElement.classList.add('text-emerald-400', 'font-bold', 'opacity-100');
                quizScore++; setTxt('current-score', quizScore);
            } else {
                btnElement.classList.replace('bg-slate-800', 'bg-rose-900'); btnElement.classList.replace('border-slate-600', 'border-rose-500'); btnElement.classList.add('text-rose-400', 'opacity-100');
                if(allBtns && allBtns[qData.answer]) {
                    allBtns[qData.answer].classList.replace('bg-slate-800', 'bg-emerald-900'); allBtns[qData.answer].classList.replace('border-slate-600', 'border-emerald-500'); allBtns[qData.answer].classList.add('text-emerald-400', 'opacity-100');
                }
            }

            // Show Explanation
            const expBox = document.getElementById('explanation-box');
            if (expBox) {
                expBox.classList.remove('hidden');
                if (isCorrect) {
                    expBox.className = 'mt-4 p-4 rounded-lg text-sm border border-emerald-200 bg-emerald-50 text-emerald-900';
                    expBox.innerHTML = `<strong>✅ Correct!</strong> ${qData.explanation}`;
                } else {
                    expBox.className = 'mt-4 p-4 rounded-lg text-sm border border-rose-200 bg-rose-50 text-rose-900';
                    expBox.innerHTML = `<strong>❌ Incorrect.</strong> ${qData.explanation}`;
                }
            }

            document.getElementById('next-btn-container')?.classList.remove('hidden');
        };

        window.showQuizResults = async function() {
            clearInterval(timerInterval);
            document.getElementById('quiz-question')?.classList.add('hidden'); document.getElementById('quiz-results')?.classList.remove('hidden');
            const percentage = Math.round((quizScore / activeQuizPool.length) * 100);
            
            latestResultId = Date.now();
            
            const newRecord = { 
                id: latestResultId, 
                date: new Date().toLocaleDateString(), 
                name: activeStudentData.name, 
                position: activeStudentData.role, 
                exp: activeStudentData.exp,
                score: percentage, 
                passed: percentage >= 80, 
                details: currentAttemptData, 
                comment: "",
                userId: currentUser ? currentUser.uid : 'anonymous',
                timestamp: Date.now()
            };

            // Save directly to Firestore Cloud
            if (useFirebase && currentUser) {
                try {
                    const assessmentsRef = collection(db, 'artifacts', appId, 'public', 'data', 'assessments');
                    const docRef = await addDoc(assessmentsRef, newRecord);
                    currentDocId = docRef.id;
                } catch (e) {
                    console.error("Failed to save assessment to cloud.", e);
                }
            } else {
                 console.warn("Saving to local storage (External Viewer Mode).");
                 newRecord.firestoreId = 'local_' + latestResultId;
                 currentDocId = newRecord.firestoreId;
                 localAssessments.push(newRecord);
                 localStorage.setItem('wc_assessments', JSON.stringify(localAssessments));
            }

            setTxt('result-student-name', activeStudentData.name); setTxt('final-score', `${percentage}%`);
            
            if (percentage >= 80) {
                setTxt('result-icon', '🏆'); setTxt('result-title', 'Passed!');
                setClass('result-title', 'text-3xl font-bold text-emerald-400 mb-2'); setClass('final-score', 'text-6xl font-extrabold text-emerald-400 font-mono mb-6');
            } else {
                setTxt('result-icon', '⚠️'); setTxt('result-title', 'Review Required');
                setClass('result-title', 'text-3xl font-bold text-amber-400 mb-2'); setClass('final-score', 'text-6xl font-extrabold text-amber-400 font-mono mb-6');
            }

            // Reset Comment Box UI
            const commentInput = document.getElementById('student-comment');
            if(commentInput) {
                commentInput.value = '';
                commentInput.disabled = false;
            }
            document.getElementById('save-comment-btn')?.classList.remove('hidden');
            document.getElementById('comment-saved-msg')?.classList.add('hidden');
        };

        window.saveComment = async function() {
            const commentVal = document.getElementById('student-comment')?.value.trim();
            if(!commentVal) return;
            
            if (useFirebase && currentUser && currentDocId) {
                try {
                    const docRef = doc(db, 'artifacts', appId, 'public', 'data', 'assessments', currentDocId);
                    await updateDoc(docRef, { comment: commentVal });
                    
                    document.getElementById('save-comment-btn')?.classList.add('hidden');
                    document.getElementById('comment-saved-msg')?.classList.remove('hidden');
                    document.getElementById('student-comment').disabled = true;
                } catch (e) {
                    console.error("Failed to sync comment to cloud.", e);
                }
            } else if (!useFirebase && currentDocId) {
                const idx = localAssessments.findIndex(a => a.firestoreId === currentDocId);
                if (idx > -1) {
                    localAssessments[idx].comment = commentVal;
                    localStorage.setItem('wc_assessments', JSON.stringify(localAssessments));
                    
                    document.getElementById('save-comment-btn')?.classList.add('hidden');
                    document.getElementById('comment-saved-msg')?.classList.remove('hidden');
                    document.getElementById('student-comment').disabled = true;
                }
            }
        };

        window.openIndividualReport = function(resultId, fromStudentView = false) {
            const idToLoad = fromStudentView ? latestResultId : resultId;
            if(!idToLoad) return;

            const data = localAssessments.find(r => r.id === idToLoad || r.firestoreId === idToLoad);
            if(!data) return;

            setTxt('rep-name', data.name); setTxt('rep-role', data.position); 
            setTxt('rep-exp', data.exp); setTxt('rep-date', data.date); setTxt('rep-score', `${data.score}%`);
            
            const scoreEl = document.getElementById('rep-score');
            if(scoreEl) scoreEl.className = `text-5xl font-black font-mono ${data.passed ? 'text-emerald-600' : 'text-rose-600'}`;

            // Radar Chart Data Calculation
            const topicScores = { "Fundamentals": {c:0, t:0}, "Pressure Math": {c:0, t:0}, "Detection": {c:0, t:0}, "Mechanics & Causes": {c:0, t:0}, "Action Protocol": {c:0, t:0}, "Operations": {c:0, t:0} };
            const missedTopicsMap = {};
            
            if(data.details) {
                data.details.forEach(ans => {
                    if(topicScores[ans.topic]) {
                        topicScores[ans.topic].t++;
                        if(ans.isCorrect) topicScores[ans.topic].c++;
                        else missedTopicsMap[ans.topic] = (missedTopicsMap[ans.topic] || 0) + 1;
                    }
                });
            }

            const radarLabels = Object.keys(topicScores);
            const radarDataPoints = radarLabels.map(topic => {
                return topicScores[topic].t > 0 ? Math.round((topicScores[topic].c / topicScores[topic].t) * 100) : 0;
            });

            // Draw Radar Chart
            const ctxIndivRadar = document.getElementById('indivRadarChart');
            if(ctxIndivRadar) {
                if(indivRadarChartInstance) indivRadarChartInstance.destroy();
                indivRadarChartInstance = new Chart(ctxIndivRadar.getContext('2d'), {
                    type: 'radar',
                    data: { labels: radarLabels, datasets: [{ label: 'Competency %', data: radarDataPoints, backgroundColor: 'rgba(56, 189, 248, 0.2)', borderColor: '#0284c7', pointBackgroundColor: '#0369a1', borderWidth: 2 }] },
                    options: { responsive: true, maintainAspectRatio: false, scales: { r: { angleLines: { color: '#e2e8f0' }, grid: { color: '#e2e8f0' }, pointLabels: { font: { family: 'Inter', size: 10, weight: 'bold' }, color: '#475569' }, suggestedMin: 0, suggestedMax: 100 } }, plugins: { legend: { display: false } } }
                });
            }

            const gapsList = document.getElementById('rep-gaps-list');
            const recList = document.getElementById('rep-rec-list');
            if(gapsList && recList) {
                gapsList.innerHTML = ''; recList.innerHTML = '';

                if(Object.keys(missedTopicsMap).length === 0) {
                    gapsList.innerHTML = '<li class="text-emerald-600 font-bold">No knowledge gaps identified. Perfect score.</li>';
                    recList.innerHTML = '<li>Maintain current operational readiness. Suitable for mentoring juniors.</li>';
                } else {
                    for(let topic in missedTopicsMap) {
                        gapsList.innerHTML += `<li class="flex justify-between items-center border-b border-rose-100 pb-1"><span>${topic}</span> <span class="font-bold text-rose-600 bg-white px-2 rounded shadow-sm">${missedTopicsMap[topic]} error(s)</span></li>`;
                        
                        let rec = "";
                        if(topic === "Pressure Math") rec = `Mandatory review of Pressure calculations (HP, KWM, MAASP) and formulas.`;
                        else if(topic === "Detection") rec = `Focus on differentiating false negatives (masking factors) from positive indicators.`;
                        else if(topic === "Action Protocol") rec = `Review the exact chronological sequence of the Flow Check protocol and API drills.`;
                        else if(topic === "Mechanics & Causes") rec = `Review U-Tube effect, ECD dynamics, and swabbing/surging mechanics.`;
                        else rec = `Review general principles related to ${topic}.`;
                        
                        recList.innerHTML += `<li class="flex items-start gap-2 bg-white p-2 rounded shadow-sm border border-sky-50"><span class="text-sky-500 mt-0.5">↳</span> <span>${rec}</span></li>`;
                    }
                }
            }

            let evalText = "", evalColor = "";
            if(data.score >= 90) {
                evalText = `${data.name} demonstrated exceptional competency. With ${data.exp} years of experience, they are well-suited for unsupervised well control operations and peer mentoring.`;
                evalColor = "border-emerald-500 bg-emerald-50 text-emerald-900";
            } else if(data.score >= 80) {
                evalText = `${data.name} demonstrated satisfactory operational competency and passed the certification. Minor gaps highlighted above should be reviewed before the next operation.`;
                evalColor = "border-sky-500 bg-sky-50 text-sky-900";
            } else {
                if(data.exp > 5) {
                    evalText = `Despite ${data.exp} years of experience, ${data.name} failed to meet the minimum competency threshold. A critical refresher on modern procedures and calculations is required to break bad habits.`;
                } else {
                    evalText = `Fundamental knowledge gaps identified. ${data.name} requires comprehensive retraining and supervised shadow-drilling before assuming active well control duties.`;
                }
                evalColor = "border-rose-500 bg-rose-50 text-rose-900";
            }

            const evalBox = document.getElementById('rep-eval-box');
            if(evalBox) {
                evalBox.className = `p-4 rounded-lg border-l-4 text-sm font-medium ${evalColor}`;
                evalBox.textContent = evalText;
            }

            setTxt('stat-total-students', results.length);

                setTxt('stat-avg-score', "0%"); setTxt('stat-group-eval', "Awaiting data to evaluate group competency...");

                const commentsList = document.getElementById('instructor-comments-list');
                if (commentsList) commentsList.innerHTML = '<p class="text-slate-400 italic text-sm text-center py-4">No comments recorded yet.</p>';
                window.renderGroupCharts(0, 0, {labels:[], data:[]}); return;
            }

            const totalScore = results.reduce((sum, r) => sum + r.score, 0); const avgScore = Math.round(totalScore / results.length);
            setTxt('stat-avg-score', `${avgScore}%`);

            let passes = 0, fails = 0; results.forEach(r => { if(r.passed) passes++; else fails++; });

            // Group Gap Analysis
            const topicScores = { "Fundamentals": {c:0, t:0}, "Pressure Math": {c:0, t:0}, "Detection": {c:0, t:0}, "Mechanics & Causes": {c:0, t:0}, "Action Protocol": {c:0, t:0}, "Operations": {c:0, t:0} };

                    r.details.forEach(ans => { 
                        if(topicScores[ans.topic]) {
                            topicScores[ans.topic].t++;
                            if(ans.isCorrect) topicScores[ans.topic].c++;
                        }
                        if(!ans.isCorrect) missesMap[ans.topic] = (missesMap[ans.topic] || 0) + 1; 
                    }); 
                } 
            });
            
            let groupEvalText = "";
            if(Object.keys(missesMap).length > 0) {
                let maxMisses = 0, weakestTopic = "";
                for (const [topic, misses] of Object.entries(missesMap)) { if (misses > maxMisses) { maxMisses = misses; weakestTopic = topic; } }

                        </tr>`;
                });
            }

            // Render Comments List
            const commentsList = document.getElementById('instructor-comments-list');
            if(commentsList) {
                commentsList.innerHTML = '';
                const commentsData = [...results].filter(r => r.comment && r.comment.trim() !== "");
                if(commentsData.length === 0) {
                    commentsList.innerHTML = '<p class="text-slate-400 italic text-sm text-center py-4">No comments recorded yet.</p>';
                } else {
                    commentsData.sort((a,b) => b.id - a.id).forEach(r => {
                        commentsList.innerHTML += `
                            <div class="bg-white p-4 rounded-lg border border-slate-200 shadow-sm relative">
                                <div class="absolute top-4 right-4 text-xs font-bold ${r.passed ? 'text-emerald-500 bg-emerald-50' : 'text-rose-500 bg-rose-50'} px-2 py-1 rounded border ${r.passed ? 'border-emerald-200' : 'border-rose-200'}">${r.score}%</div>
                                <div class="mb-2">
                                    <strong class="text-slate-800">${r.name}</strong> <span class="text-slate-500 text-xs ml-2">${r.date}</span>
                                    <span class="block text-xs text-sky-600 font-medium">${r.position} (${r.exp} yrs)</span>
                                </div>
                                <p class="text-sm text-slate-700 bg-slate-50 p-3 rounded border border-slate-100 italic">"${r.comment}"</p>
                            </div>
                        `;
                    });
                }
            }

            window.renderGroupCharts(passes, fails, topicData);
        };

        window.openGroupReport = function() {
            const results = localAssessments;
            if(results.length === 0) return alert("No data available for group report.");

            const total = results.length;
            const passes = results.filter(r => r.passed).length;
            const avgScore = Math.round(results.reduce((sum, r) => sum + r.score, 0) / total);
            const passRate = Math.round((passes / total) * 100);

            setTxt('grep-total', total);
            setTxt('grep-passrate', `${passRate}%`);
            setTxt('grep-score', `${avgScore}%`);

            const scoreEl = document.getElementById('grep-score');
            if(scoreEl) scoreEl.className = `text-5xl font-black font-mono ${avgScore >= 80 ? 'text-emerald-600' : 'text-rose-600'}`;

            const topicScores = { "Fundamentals": {c:0, t:0}, "Pressure Math": {c:0, t:0}, "Detection": {c:0, t:0}, "Mechanics & Causes": {c:0, t:0}, "Action Protocol": {c:0, t:0}, "Operations": {c:0, t:0} };
            const missedTopicsMap = {};
            
            results.forEach(r => {
                if(r.details) {
                    r.details.forEach(ans => {
                        if(topicScores[ans.topic]) {
                            topicScores[ans.topic].t++;
                            if(ans.isCorrect) topicScores[ans.topic].c++;
                            else missedTopicsMap[ans.topic] = (missedTopicsMap[ans.topic] || 0) + 1;
                        }
                    });
                }
            });

            const gapsList = document.getElementById('grep-gaps-list');
            const recList = document.getElementById('grep-rec-list');
            if(gapsList && recList) {
                gapsList.innerHTML = ''; recList.innerHTML = '';

                if(Object.keys(missedTopicsMap).length === 0) {
                    gapsList.innerHTML = '<li class="text-emerald-600 font-bold">No knowledge gaps identified. Perfect score for the fleet.</li>';
                    recList.innerHTML = '<li>Maintain current operational readiness. No immediate group training required.</li>';
                } else {
                    const sortedTopics = Object.entries(missedTopicsMap).sort((a, b) => b[1] - a[1]);
                    sortedTopics.forEach(([topic, count]) => {
                        gapsList.innerHTML += `<li class="flex justify-between items-center border-b border-rose-100 pb-1"><span>${topic}</span> <span class="font-bold text-rose-600 bg-white px-2 rounded shadow-sm">${count} error(s)</span></li>`;
                        
                        let rec = "";
                        if(topic === "Pressure Math") rec = `Conduct a dedicated workshop on Pressure calculations (HP, KWM, MAASP).`;
                        else if(topic === "Detection") rec = `Run simulator drills focusing on differentiating false negatives (masking factors) from positive indicators.`;
                        else if(topic === "Action Protocol") rec = `Schedule rig-floor drills to review the chronological sequence of the Flow Check protocol and API drills.`;
                        else if(topic === "Mechanics & Causes") rec = `Review U-Tube effect, ECD dynamics, and swabbing/surging mechanics.`;
                        else rec = `Review general principles related to ${topic}.`;
                        
                        recList.innerHTML += `<li class="flex items-start gap-2 bg-white p-2 rounded shadow-sm border border-sky-50"><span class="text-sky-500 mt-0.5">↳</span> <span>${rec}</span></li>`;
                    });
                }
            }

            let evalText = "", evalColor = "";
            if(avgScore >= 80) {
                evalText = `The fleet demonstrates satisfactory operational competency. Minor gaps highlighted above should be reviewed.`;
                evalColor = "border-sky-500 bg-sky-50 text-sky-900";
            } else {
                evalText = `Fundamental knowledge gaps identified across the fleet. Comprehensive retraining required before assuming active well control duties.`;
                evalColor = "border-rose-500 bg-rose-50 text-rose-900";
            }

            const evalBox = document.getElementById('grep-eval-box');
            if(evalBox) {
                evalBox.className = `p-4 rounded-lg border-l-4 text-sm font-medium ${evalColor}`;
                evalBox.textContent = evalText;
            }

            document.getElementById('group-report-modal')?.classList.remove('hidden');
        };

        window.closeGroupReport = function() { document.getElementById('group-report-modal')?.classList.add('hidden'); };

        window.clearResults = async function() { 
            if(confirm("Delete all records?")) { 
                if(useFirebase && currentUser) {
                    try {
                        const assessmentsRef = collection(db, 'artifacts', appId, 'public', 'data', 'assessments');
                        const snapshot = await getDocs(assessmentsRef);
                        const deletePromises = snapshot.docs.map(docSnap => deleteDoc(doc(db, 'artifacts', appId, 'public', 'data', 'assessments', docSnap.id)));
                        await Promise.all(deletePromises);
                        alert('Cloud database cleared.');
                        window.updateInstructorDashboard();
                    } catch(e) {
                        console.error('Error clearing database', e);
                        alert('Failed to clear database. Ensure you have permissions.');
                    }
                } else {
                    localStorage.removeItem('wc_assessments');
                    localAssessments = [];
                    alert('Local database cleared.');
                    window.updateInstructorDashboard();
                }
            } 
        };

            }).join("\n");
            
            const link = document.createElement("a"); link.href = encodeURI("data:text/csv;charset=utf-8," + csv); link.download = `Group_Analytics.csv`;
            document.body.appendChild(link); link.click(); document.body.removeChild(link);
        };
